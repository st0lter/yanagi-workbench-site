Yanagi Workbench

Python Automation Made Easy

> What is the Yanagi Workbench?

This project is designed to be a general tool to be used in many scenarios, be it in administrative tasks or in most complex workflows, with just a few clicks.

The main idea is to gather every essential tool in a single application, from the most basic activities to the most complex tasks.

> For what cases should I use it?

Do you need to save some files? No worries!

Do you need to convert from PDF to DOCX and/or vice-versa? Totally possible!

Do you need to schedule a task that needs to be done daily, weekly or monthly? No problem at all!

Do you need to create a web automation for testing? That's also possible!


And the best thing? The project receives constant updates and it is for free!

> What tools does it use?

It is entirely made with Python along with the usage of its many libraries.

  • CustomTkinter
  • Pillow
  • Selenium
  • Schedule
